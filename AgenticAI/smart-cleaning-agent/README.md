# Smart Cleaning Agent Simulation

This project demonstrates a Simple Reflex Agent that operates in a 2x2 environment.

## Description
- The environment consists of 4 rooms.
- Each room can be Clean or Dirty.
- The agent senses the current room.
- If the room is Dirty, it cleans it.
- Otherwise, it moves to the next room.

## Tools Used
- Python
- Matplotlib
